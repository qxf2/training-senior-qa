We expect this exercise to take you somewhere between 15-minutes to 1 hours. If you take more than one hour, please post what you tried along wtih error messages to our skype channel.

We will be using git bash quite a lot at Qxf2. Therefore, for this tutorial, you are going to install the latest version from the official site.

In keeping with our attempt to help you think in different ways, we have listed three approaches that work. There is no shame or pride in choosing anyone of them. Just choose the method you are comfortable with and proceed.

 

Approach 1: Try it out yourself

  1. We'd like to promote independent thinking and problem solving. So if you feel ready, try to install Git Bash without referring to any documentation, go for it!

 

Approach 2: Google for instructions

  1. Google for Git Bash install 'your operating system name'
  2. Follow the instructions

 

Approach 3: follow instructions

  1. Visit https://gitforwindows.org/index.html
  2. Click on the Downloads
  3. On the release page, choose latest release for your operating system
  4. Download the executable and follow the steps
  5. Once installation completes, Press windows key and search for Git bash and open it

 

Exercise

Run the below commands to configure your git, so that these details would get associated with any commits you create in future . git config --global user.name "Your name" . git config --global user.email "Your qxf2 emaild id"

 

Verify your setup

After you are done with the exercise listed above, it is time to verify your Git Bash installation. It is a good habit to take small steps and verify them immediately. This will help you avoid getting stuck later on. To verify your setup, pull up your terminal prompt and run the command git --version. You should see the version you installed listed.
