Welcome to Qxf2 Trainer - this screen is the start of your journey towards becoming a more technical tester. This training program has been created by testers for other testers. We have made many tweaks to our training program to incease the odds that you actually stick to this program.

SETUP

There are many other nice features to our program that you will discover as you work through our exercises. But before you begin, please follow all the instructions in the setup section of this page. Don't be scared if you have tried previously and failed. We wrote the setup guides assuming things would go wrong and that you would need help troubleshooting problems.

In this section, you will be guided to setup your environment. Proceed in order. Click on the first link/task that you are yet to complete.

  2. Setup Visual Studio Code
  3. Setup webdriver for Chrome
  4. Setup git bash
  5. Sign up for a free WordPress account

EXERCISES

We have tried our best to keep these exercises relevant and creative. Unlike most courses, we will not steadily build up your theoretical knowledge and then give you a practical problem. Instead, we will steadily present you with practical (and useful!) problems and hope that your theoretical knowledge increases with practice. For example, we won't teach you Python from scratch but will give you enough help to modify Python code based on patterns you observe.

  1. Clone a GitHub repository and run an automates test
  2. Edit the above test to take values that suit you
  3. Publish your resume on your WordPress site
  4. Contribute to open source by fixing an issue on GitHub
  5. Fetch website data without using a browser
  6. Hook up your automation with a test case manager and a CI tool
