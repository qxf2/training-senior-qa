We expect this exercise to take you somewhere between 15-minutes to 2 hours. If you take more than two hours, please post what you tried along wtih error messages to our skpye channel.

In keeping with our attempt to help you think in different ways, we have listed three approaches that work. There is no shame or pride in choosing anyone of them. Just choose the method you are comfortable with and proceed.

 

Approach 1: Try it out yourself

  1. We'd like to promote independent thinking and problem solving. So if you feel ready, try to install Visual Studio Code for Python without referring to any documentation, go for it!

 

Approach 2: Google for instructions

  1. Google for Visual Studio Code for Python install 'your operating system name'
  2. Follow the instructions

 

Approach 3: follow instructions

  1. Visit https://code.visualstudio.com/download to install Visual Studio Code for your operating system
  2. Visit https://marketplace.visualstudio.com/items?itemName=ms-python.python to install Python extension for Visual Studio Code
  3. Start Python coding on Visual Studio Code

 

Exercise

Once you are done, create a new file using Visual Studio Code, write print "Hello my name is YOUR_NAME" in it and save the file. Make sure there are no spaces in front of the 'p' of print. Give it a name my_first_step.py. Now figure out how to execute the file using Visual Studio Code.

 

Verify your setup

After you are done with the exercise listed above, try writing down what you just acheived in 2-3 sentences. Recapping and verbalizing what you just did is an under-rated skill when you are learning something new. So don't skip this step! By the way, if you are new to coding, you took what looks like one small step for a programmer but it was (to paraphrase the famous Moon quote) a giant leap for you!
